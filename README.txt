== Description ==

The International Address Checker is a free (WooCommerce) plugin that guarantees correct address input from your customers on, for example, your own registration or checkout webpage. It covers more than 170 countries and comes in the correct address structure, as applicable in that particular country. 

=== HOW DOES IT WORK? ===

By simply implementing the plugin into your own website, you are assured of fast and error-free address entries of customers worldwide. A link to Matrixianâ€™s international database checks the address input during a registration process or online purchase. If the address does not exist or when it is written incorrectly, the International Address Checker plugin immediately provides your customer with the correct alternative!

For technical details about the API click [here](https://platform.matrixiangroup.com/api/documentation/addresschecker "Matrixian API documentation").

=== PLUGIN ACTIVATION ===

1. Create a free account on the [Matrixian platform](https://platform.matrixiangroup.com/register?scenario=iacPlugin "Matrixian Platform")
2. Go to your Wordpress account and click on Woocommerce > Settings > Advanced > International Address Checker
3. Fill in your Matrixian Platform account credentials
4. Save changes
5. The International Address Checker is now active

== Installation ==

1. Upload `matrixian-address-validator.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Settings page and enable the API
4. Fill out your Matrixian account credentials
5. Address Validator is now active

We provide support for the following countries at the moment: Austria, Belgium, Czech Republic, Denmark, Germany, France, Italy, Luxembourg, Spain, Sweden, The Netherlands, United Kingdom, United States

No, this plugin is used for address validation purposes only.
